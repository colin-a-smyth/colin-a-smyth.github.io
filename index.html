Background Information on WikiSpeedia

WikiSpeedia is a game where the player is given a starting Wikipedia article and a finishing Wikipedia article, and the aim is to get from start to finish using as few hyperlinks as possible. You can play the game here.

The Data

The data that we have from the game is as follows:

  • articles.tsv
    • Containing all navigable articles in the game.
  • categories.tsv
    • The categorz and sub-categories that each article falls into.
  • links.tsv
    • Containing all links between articles. It should be noted that this set is directed, A -> B does not imply B -> A.
  • paths_finished.tsv
    • Containing all paths that were finished by players in the game.
  • paths_unfinished.tsv
    • Containing all paths that were unfinished by players in the game.
  • shortest-path-distance-matrix.txt
    • A matrix containing the shortest distances between all articles.
We decided to investigate if there was a bias towards certain countries or world regions in both the articles that the game is based on, and subsequently how people played the game.
